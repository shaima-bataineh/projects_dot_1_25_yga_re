# SalesFlow Project

This project was built using **Next.js (App Router)**.

## Description
A simple sales management website that demonstrates:
- Static Rendering
- Dynamic Rendering (SSR)
- Reusable components
- Clean project structure

## Features
- Home page with reusable components
- Services page using Server-Side Rendering
- API routes for handling data
- Responsive layout
